Seletores CSS

Bem-vindo!

No CSS nós possuimos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML especifica.
  • Seletor ID(#): Busca elementos através do atributo "id".
  • Seletor Classe (.): Busca elementos através do atributo "Class".
  • Seletor Universal (*): Seleciona todos os elementos HTML.
  • Seletor Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico.
